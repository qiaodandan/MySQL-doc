## 本地开发的配置
我用的方法是clone直接去下载这个项目, 这是最常用下载或拉取github仓库的方法.
```shell
$ git clone https://github.com/qiaodandan/MySQL-doc.git
```
## 具体开发操作
```shell
$ cd MySQL-doc/
$ ls
$ touch github基本命令.md(可以touch多个文件)
$ git status
$ git add .
$ git status
$ git commit
$ ls
$ git branch -av
$ git pull origin master
$ git push origin master
```
